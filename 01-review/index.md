# First Semester Review

## Overview of job roles

- Programmers build the IT backend and middleware templates that generate and publish the HTML content.
- Visual designers create mockups that programmers use to build the appearance of the web site or app
- Designers need to understand how CSS works in order to build designs that look and work like web sites
- Designers who know some code have a competitive advantage over designs who don't code.
