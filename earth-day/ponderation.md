# Earth Day Project

## Evaluation Criteria

| Criteria | Grade |
|--------|-------|
| Custom Fonts<br>Use Google or Adobe fonts  |  /5   |
| Use of Background Images<br>Decorate elements with CSS background images only  |  /5   |
| Use of Pseudo-Elements for Background Images<br>Add additional imagery before/after elements<br>Minimum one instance |  /10  |
| CSS Grid Layout  |  /20  |
| Element Inner Layout Using FlexBox   |  /10  |
| HTML Picture tag: Orientation<br>Picture must change from portrait to landscape<br>Applies only to content images (img tag), but not UN icons |  /10  |
| Typography | /15   |
| Overall Aesthetics | /25   |
| Grand Total | /100  |
