Introduction to multicolumn page layout with floats

Example 1: figure tags are floated left, no hacks, broken box model layout

figure {float:left; margin: 0.5rem;}

Example 2: with clearfix added to parent article tag

<article class="clearfix">

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}

Example 3, with overflow: auto; added to parent article tag

article {overflow: auto;}
