For demo purposes, the images below are being forced to display as block level elements.

In web design, most elements that are laid out on a page, such as: divs, sections, paragraphs, list, list items, etc, all display as block level elements. Meaning that they stack one on top of the other vertically. In order to keep this demo simple, we are using images instead of code. Images by default are inline elements. In order to practice standard page layout techniques such as float, flexbox and css grid, we are forcing the images to behave like blocks so we can pretend we are laying out standard page building elements.
